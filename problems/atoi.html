ATOI

Problem Statement

Implement the myAtoi(string s) function, which converts a string to a 32-bit signed integer.

The algorithm for myAtoi(string s) is as follows:

  1. Ignore any leading whitespace (" ").
  2. Determine the sign by checking if the next character is '-' or '+', assuming positivity if neither is present.
  3. Read the integer by skipping leading zeros until a non-digit character is encountered or the end of the string is reached. If no digits were read, then the result is 0.
  4. If the integer is out of the 32-bit signed integer range [-231, 231 - 1], then round the integer to remain in the range. Specifically, integers less than -231 should be rounded to -231, and integers greater than 231 - 1 should be rounded to 231 - 1.
  5. Return the integer as the final result.
            
                Example 1:
                Input: s = "42"
                Output: 42

                Example 2:
                Input: s = "   -42"
                Output: -42

                Example 3:
                Input: s = "4193 with words"
                Output: 4193

Solution

            
                class Solution:
                    def myAtoi(self, s: str) -> int:
                        s = s.strip()
                        
                        if not s:
                            return 0
                        
                        res = 0
                        sign = 1
                        idx = 0
                        
                        if s[idx] == "-":
                            sign = -1
                            idx = 1
                        elif s[idx] == "+":
                            idx = 1
                            
                        for i in range(idx, len(s)):
                            if s[i].isalpha() or not s[i].isalnum():
                                break
                            res = res * 10 + (ord(s[i]) - ord("0"))
                            
                        res = res * sign
                            
                        INT_MIN, INT_MAX = -2**31, 2**31 - 1
                        
                        if res < INT_MIN:
                            return INT_MIN
                        
                        if res > INT_MAX:
                            return INT_MAX
                        
                        return res
